BIO

Qraft

With a time frame of 7 days, I created Qraft - a full stack web application for furniture makers to showcase DIY or handmade pieces. A 'qraft' is a moniker for pieces of furniture uploaded by the user for the site.

Users who are logged in can access more of the app's functionality including the ability to post, edit or delete qrafts, and to leave comments about pieces of furniture.

Software:

JavaScript | React | Webpack | Python 3.7 | Flask | Marshmallow | PostgreSQL | SQLAlchemy | Insomnia | React Semantic UI |
