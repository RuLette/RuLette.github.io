PROJECT ONE

Personal Space Invaders

Personal space invaders is a solo project built over the time frame of 7 days. This 2D, single-player game inspired by the classic arcade game, Space Invaders.

The objective of Space Invaders is for the player to move a spaceship across a screen and shoot descending swarms of aliens. If the player has been hit by an enemy laser or if the enemy aliens reach the bottom of the screen, it's game over.

Software:

JavaScript (ES6) | jQuery | HTML | HTML5 audio | SCSS | CSS Animations | GitHub

PROJECT TWO

Dayout

Dayout is a 2 day solo project that shows the user festivals that are currently ongoing in London. It also displays the current weather and tube service status. Upon selecting an event, the user may read its description while details of the venue are presented on a map.

Software:

React | Axios | Webpack | Insomnia | Dotenv | Moment | Bulma

PROJECT THREE

Booker

With a time frame of 7 days, we built a full-stack application for users to share books in the community in a team of four. The project utilised React in the front end, with MongoDB as a database in the back end.

On this project my role involved building backend models and controllers for books, user profile components, book schema testing, book forms and adding error handlers to form components.

Software:

React | Axios | CSS | Webpack | Insomnia | Dotenv | Moment | Bulma | MongoDB | Mongoose | Express |

PROJECT FOUR

Qraft

With a time frame of 7 days, I created Qraft - a full stack web application for furniture makers to showcase DIY or handmade pieces. A 'qraft' is a moniker for pieces of furniture uploaded by the user for the site.

Users who are logged in can access more of the app's functionality including the ability to post, edit or delete qrafts, and to leave comments about pieces of furniture.

Software:

JavaScript | React | Webpack | Python 3.7 | Flask | Marshmallow | PostgreSQL | SQLAlchemy | Insomnia | React Semantic UI |
